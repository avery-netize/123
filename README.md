# Discord role guard bot (Python)

Бот блокирует выдачу ролей, если модератор пытается выдать роль, которая ему не разрешена.

## 1) Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 2) Настройка

1. Скопируй `.env.example` в `.env` и заполни:
   - `DISCORD_TOKEN`
   - `GUILD_ID`
   - `LOG_CHANNEL_ID` (канал для логов)
2. Можно настроить правила через slash-команду в Discord:
   - `/настройка_ролей` -> `какая_роль` (какая роль выдает), `какую_роль` (какую роль может выдать)
   - `/ресет_роли` -> `какая_роль` (удалить все правила выдачи у выбранной роли)
   - `/настройка_создания_ролей` -> 2 графы: `кто_может` и `кто_не_может`
   - `/ресет_создания_ролей` -> `какая_роль` (убрать у выбранной роли доступ на создание ролей)
   - `/правила_ролей` -> показать все текущие правила
3. Либо вручную заполняй `role_policy.json`.

Формат `role_policy.json`:

```json
{
  "ALLOWED_GIVERS": {
    "ID_РОЛИ_КТО_ВЫДАЕТ": [ID_РОЛИ_КОТОРУЮ_МОЖНО_ВЫДАТЬ, ID_ДРУГОЙ_РОЛИ]
  },
  "ALLOWED_ROLE_CREATORS": [ID_РОЛИ_КОТОРАЯ_МОЖЕТ_СОЗДАВАТЬ_РОЛИ],
  "BLOCKED_ROLE_CREATORS": [ID_РОЛИ_КОТОРАЯ_НЕ_МОЖЕТ_СОЗДАВАТЬ_РОЛИ]
}
```

## 3) Важные права бота

- Включи `SERVER MEMBERS INTENT` в Discord Developer Portal.
- Дай боту права:
  - `Manage Roles`
  - `View Audit Log`
- Роль бота в иерархии должна быть **выше** ролей, которые он должен снимать.

## 4) Запуск

```bash
python bot.py
```
